# IHSG Transaction Log

A simple stock tracker for Indonesian Stock Exchange (IDX).

## Features
- Track IHSG movement
- Log buy & sell transactions
- Simple portfolio tracking
- Lightweight web interface

## Tech Stack

- JavaScript
